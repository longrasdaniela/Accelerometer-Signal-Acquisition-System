<h1> 👩‍💻 Accelerometer Signal Acquisition System</h1>

<h2>Description</h2>
This project is a complete embedded system designed for the real-time acquisition, processing, and visualization of triaxial acceleration data. The system integrates an analog accelerometer with a PIC microcontroller, transmitting data via serial communication to a graphical user interface (GUI) developed in MATLAB.
<br />

<h2>Languages and Utilities Used</h2>

- **Firmware:** C and Assembly languages</b> 
- **IDE / Compiler:** MPLAB X IDE & XC8 Compiler</b> 
- **Microcontroller:** Microchip PIC18F47Q10 </b> 
- **Processing & GUI:** MATLAB </b> 
- **Communication Protocol:** RS232 (UART) via an FTDI/Serial-to-USB converter </b> 
- **Key Hardware:** MMA7361L Analog Triaxial Accelerometer (NXP/Freescale) </b>
